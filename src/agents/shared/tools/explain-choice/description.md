# Explain Choice Tool

## Purpose

Provides detailed explanations for exercise selection, programming choices, or
routine structure decisions.

## When to Use

- When user asks "why" about any programming decision
- To provide scientific backing for exercise choices
- To educate users about training principles
- When explaining rep ranges, frequencies, or volume recommendations

## Input

- **topic**: What to explain (exercise choice, rep range, frequency, etc.)
- **reasoning**: Detailed explanation of the reasoning behind the choice
- **evidence**: Optional scientific evidence or principles supporting the choice

## Output

Returns an explanation object with structured reasoning that helps users
understand the science and logic behind programming decisions.
